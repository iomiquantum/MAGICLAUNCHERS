Bienvenido al Orchestrator

1
Lanza agentes
Crea sesiones con el modelo que necesites. Cada una es independiente. Acepta el trust dialog la primera vez en terminal.
2
Envia tareas
Usa el Centro de Tareas o la tarea rapida en cada tarjeta. Broadcast envia a todas. Multi-select a las que elijas.
3
Monitorea en vivo
Clic en el preview para vista completa tipo terminal. Notas para anotar, historial de tareas, y cola para encolar trabajo.
?
Que modelo usar
Opus = lo mas potente. Sonnet = coding diario. Haiku = rapido y barato. Orquestador = decide solo.
$
Costos estimados
Opus ~$0.08/min, Sonnet ~$0.02/min, Haiku ~$0.005/min. Son estimaciones, el costo real depende del uso de tokens.
>
Terminal directa
tmux attach -t NOMBRE en terminal. Salir sin cerrar: Ctrl+B luego D. Instala como app: menu Chrome > Instalar.
